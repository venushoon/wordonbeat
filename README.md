자동 단계 넘기기 구현 (단계당 6초): 각 단계가 시작되면 6초 타이머가 작동합니다 (useEffect와 setTimeout 사용). 상단 UI에 남은 시간을 표시하며, 6초가 지나면 자동으로 다음 단계로 넘어갑니다. 마지막 단계 끝나면 '미션 성공' 화면으로 이동합니다.

그림/단어 제시 옵션 추가: 대기 화면에 옵션 선택 버튼을 추가했습니다 (cardDisplayMode 상태). '그림으로만' 또는 '그림과 단어로' 중 선택할 수 있으며, 게임 플레이 중 카드를 뒤집었을 때 해당 옵션에 맞춰 내용이 표시됩니다.

미션 실패 로직 및 다시 시작 버튼: 잘못된 카드 (예: "하천"이 아닌 카드)를 뒤집으면 '실수' 카운트가 증가합니다. Mistakes가 3번이 되면 '미션 실패' 화면으로 이동합니다. 실패 화면, 성공 화면, 그리고 게임 플레이 중 하단 컨트롤 바에 '다시 시작' 버튼을 추가하여 언제든지 대기 화면으로 초기화할 수 있게 했습니다.
